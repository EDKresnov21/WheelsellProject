# Wheelsell

Wheelsell is a project, given to me by my father and his friend to test my coding skills and keep them "maintained" in a way during the summer break
It is a car selling website. It has full moderation system and an advanced searching system.

## What you'll need to boot it up

**SQL Server Express** *(and something to operate it, like MSSMS)*
**.NET 9 SDK**
**Node.js**

## How to set Wheelsell up

Run 'setup.bat'

## How to boot Wheelsell up

Start 'launch.bat'. Then start 'webapp.bat' and, when it prints out a link, open it in your browser.

## What I didn't do

Unfortunately, I didn't manage to create a designated Gmail account for the project, so it doesn't have email verification

## How to set Wheelsell up manually

First of all, you will need **SQL Server Express** installed on your computer, as well as some software designed to use it - I recommend Microsoft SQL Server Management Studio (MSSMS).
When downloaded, head over to 'DATABASE SCRIPT.sql', open it in MSSMS. 
First, select only the first line (CREATE DATABASE Wheelsell;) and press F5. 
Then select everything from line 2 onwards and press F5 again.
